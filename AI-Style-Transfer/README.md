## 项目简介
基于TensorFlow Lite轻量化深度学习模型，实现实时图像艺术风格迁移，属于计算机视觉前沿AI应用。

## 核心特性
- 可迁移：模型/代码跨平台部署
- 可优化：提升推理速度、增强风格效果
- 可调参：风格强度、图像尺寸、帧率自由控制
- 前端展示：固定显示学号+姓名

## 运行方法
1. pip install -r requirements.txt
2. 下载model.tflite放入根目录
3. python app.py
4. 访问 http://127.0.0.1:5000

## 项目介绍
本项目基于Google Magenta前沿轻量化深度学习模型，实现任意图像艺术风格迁移，属于计算机视觉热门方向。

## 功能特点
- 可迁移：模型可部署至网页、服务器、移动端
- 可优化：支持模型量化、推理加速、效果增强
- 可调参：可自由调节风格强度参数
- 前端展示：页面显示学号与姓名

## 运行方式
1. 安装依赖：pip install -r requirements.txt
2. 运行服务：python app.py
3. 打开浏览器访问：http://127.0.0.1:5000
